css 작성방법 3가지

1. 인라인 스타일시트. 태그의 style속성으로 css를 작성
2. 내부 스타일시트 : html과 head 태그에 style태그를 생성해 css를 생성
3. 외부스타일시트 : 외부css 파일에 css코드를 작성하고 gead에 link를 통해 문서에 css를 포함시키는 방싱 
        * 스타일 우선 순위
        인라인스타일시트 >>> 내부스타일시트 >>> 외부스타일시트

        css 파일에 작성한 css 내용과, 내용스타일시트로 작성한 css내용이 충돌한 다음 내부스타일시트로 적용
        내부스타일시트로 작성한 css내용과 인라인스타일시트로 작성한 css내용이 충돌한다면 인라인스타일시트로 적용
